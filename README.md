# edcrypt &mdash; AES256 Encrypt/Decrypt File CLI

A simple AES256 file encryption and decryption CLI.

# Installation

    npm install edcrypt -g

# Usage

    $ edcrypt e file_name.txt password

    $ edcrypt d file_name.txt.enc password
